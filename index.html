FAQ

Common questions.

Do I need special hardware for the gate system?

Most car washes already have a barrier controller. We integrate via API with common systems (IoT relays, LPR cameras, standard access controllers). If you need new readers, we can recommend compatible hardware — typically $800–$2,500 per lane depending on setup.

What if we open before the app is ready?

We prioritize a "soft launch" core — membership sign-up and admin tools — by week 5, so you can start enrolling members even while gate integration is being finalized.

Can we change membership plans later?

Yes. Your admin dashboard lets you create, edit, and retire plans without developer involvement. Price changes sync to Stripe automatically.

Who owns the code?

You receive full ownership of all custom code upon final payment — no licensing fees or recurring platform charges.

What about App Store fees?

Apple and Google developer accounts (~$99/yr and $25 one-time) are standard industry costs and can be set up under your business name at launch.
